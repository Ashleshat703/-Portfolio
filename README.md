## Features

- Personal portfolio landing page
- Animated rotating text effect
- Social media links
- Clean and simple UI
- Frontend-only implementation

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Typed text animation

## Project Structure

```text
CodeClause_project_portfolio/
|-- index.html
|-- style.css
|-- port.js
|-- images and media assets
````

### How It Works
The homepage introduces the portfolio owner
JavaScript rotates text roles dynamically
CSS styles the layout and visual presentation
Links connect to external social and profile pages


### Getting Started
Clone or download this repository
Open the project folder
Open index.html in any browser
